# First Repo

github과 git을 배우기 위해 태어난 나의 첫 저장소.

See [Demo](https://www.google.com/)

## Prerequisites

- Python^3.12
- Pandas==1.0.0

## How to Start

```shell
% git clone {repo addr}
% cd {repo addr}
% pip install -r requirements.txt
% python main.py
```

## Installation

### github clone

`% git clone {repo addr}`

### Using Homebrew(for MacOS)

`% brew install {pkg name}

## Features

- 숨쉬기
- 계산하기
- 전력 먹기
